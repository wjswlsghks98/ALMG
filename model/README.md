# File Structure

```
|
|- src
    |- RTC.py            : Script for training CNN/LSTM network of Detection and Classification framework
|
|- train
    |- Classification   
           |- data       : Extracted data that is used for training classification network
           |- itvs       : Labels
           |- model_CNN  : Trained models
    |
    |- Detection
           |- data       : Extracted data that is used for training detection network
           |- itvs       : Labels
           |- model_CNN  : Trained models
   
```
